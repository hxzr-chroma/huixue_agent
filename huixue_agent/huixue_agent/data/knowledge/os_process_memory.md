# 操作系统学习要点摘录（知识库示例）

## 进程与线程

进程是资源分配的基本单位，线程是 CPU 调度的基本单位。同一进程内的多个线程共享地址空间与打开的文件描述符，切换开销通常小于进程切换。多线程编程需注意同步与互斥，常见原语包括互斥锁、信号量、条件变量。

## 进程调度

常见调度算法包括先来先服务（FCFS）、最短作业优先（SJF）、时间片轮转（RR）以及多级反馈队列。交互式系统往往结合时间片与优先级，在响应时间与吞吐量之间折中。

## 死锁

死锁的四个必要条件：互斥、占有且等待、不可抢占、循环等待。预防可从破坏某一条件入手；避免可使用银行家算法；检测与恢复则允许死锁发生后再处理。

## 内存管理

逻辑地址到物理地址的映射可通过分页、分段或段页式实现。虚拟内存利用页表与缺页中断，把部分页面换出到磁盘，从而提供比物理内存更大的逻辑地址空间。

## 页面置换

常见置换算法包括 FIFO、LRU、Clock 等。LRU 近似“最近最少使用”，通常比 FIFO 更能减少缺页率，但实现成本更高。
